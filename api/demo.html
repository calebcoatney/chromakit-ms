Example API Call

Here's how a web frontend would process a chromatogram:

const response = await fetch('http://localhost:8000/api/process', { method: 'POST', headers: {'Content-Type': 'application/json'}, body: JSON.stringify({ x: chromatogramX, y: chromatogramY, params: { smoothing: {enabled: true}, baseline: {method: 'asls', lambda: 1000000}, peaks: {enabled: true, peak_prominence: 0.05} } }) }); const result = await response.json(); console.log(`Detected ${result.peaks_x.length} peaks`);
